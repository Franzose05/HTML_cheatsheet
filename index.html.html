2. Attributes

An HTML attribute is extra information added to an HTML tag that helps define the behavior, appearance, or metadata of that element

2.1. Structure 

 <p style="color:red;"<tagname attribute="value"> Content </tagname>

2.2. href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to
Example create a Link to a YouTube video:
click me
it would look like this → 

 <a href="https://youtu.be/GBIIQ0kP15E?si=8mp70O5nA5_fN3gH"> click me </a>

2.3. src Attribute

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed
The image can be sourced localy or from a Website

local Ex: <img src="shutterstockRF_623451695.jpg">
Website Ex: <img src="https://www.lonelyplanet.de/magazin/reportagen/asien/japans-naturwunder.html">

The width and height of an image can also be changed → <img src="img_girl.jpg" width="500" height="600">

The alt attribute can be used to describe what is shown in a image, when its not displayed (img not found) 

<img src="shutterstockRF_623451695.jpg" alt="Cherry blossoms in full splendor">

2.4. style Attribute

The style attribute is used to add styles to an element, such as color, font, size, and more.

This is a red paragraph 

 <p 	style="color:red;"> This is a red paragraph </p>

4. Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more

4.1. Structure

5. Formatting

Tag What it does Example
<b> Bold text (visually bold, no extra meaning) <b>Bold</b> Bold
<strong> Important text (bold + semantic meaning) <strong>Important</strong> Important
<i> Italic text (visual only) <i>Italic</i> Italic
<em> Emphasized text (italic + semantic meaning) <em>Emphasized</em> Emphasized
<mark> Highlighted/marked text <mark>Marked</mark> Marked
<small> Smaller, less prominent text <small>Small</small> Small
<del> Deleted/struck-through text <del>Deleted</del> Deleted
<ins> Inserted text (usually underlined) <ins>Inserted</ins> Inserted
<sub> Subscript (e.g., chemical formulas) H<sub>2</sub>O H2O
<sup> Superscript (e.g., exponents) E = mc<sup>2</sup> E = mc2
